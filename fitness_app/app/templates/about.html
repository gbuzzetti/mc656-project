MC656 - Projeto de Engenharia de Software - UNICAMP (2º Semestre de 2024)

Este projeto faz parte do curso MC656 de Engenharia de Software oferecido pela Universidade Estadual de Campinas (UNICAMP) no segundo semestre de 2024. O objetivo do projeto é aplicar os conhecimentos adquiridos em sala de aula para desenvolver uma solução tecnológica inovadora que atenda a uma necessidade real da sociedade.

Descrição do Projeto

O projeto será focado na área de saúde e bem-estar, com o desenvolvimento de uma aplicação que auxilia no monitoramento da saúde pessoal. A aplicação oferecerá funcionalidades como:

  • Fornecimento de dicas nutricionais personalizadas;
  • Cálculo de índices calóricos baseados na dieta do usuário;
  • Monitoramento da quantidade recomendada de ingestão de água;
  • Relatórios sobre hábitos saudáveis e desempenho ao longo do tempo.

Com isso, busca-se promover uma melhoria na qualidade de vida e uma maior conscientização sobre hábitos saudáveis.

Objetivos

Os objetivos principais do projeto incluem:

  • Aplicar metodologias ágeis para o desenvolvimento do software;
  • Trabalhar em equipe para simular um ambiente real de engenharia de software;
  • Desenvolver um produto funcional e de fácil usabilidade;
  • Garantir a qualidade do software por meio de testes abrangentes e feedback contínuo.
